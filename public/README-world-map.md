# World Map Image Setup

To use your flat map image for the globe:

1. Save your world map PNG image to this directory (`public/`)
2. Name it `world-map-outline.png` (or update the filename in `examples/marker/main.ts`)
3. The image should be in equirectangular projection (2:1 aspect ratio, e.g., 2048x1024px)

The image will be accessible at `/world-map-outline.png` when the dev server is running.
